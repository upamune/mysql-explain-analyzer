<h3 align="center">
  <b>
    ⚡️ A cli tool for analyzing MySQL EXPLAIN
  </b>
</h3 >


`EXPLAIN FORMAT=json` で出力したEXPLAINの結果を解析するCLIツール

## Installation

```bash
$ brew install upamune/tap/mysql-explain-analyzer
```

## Usage

```bash
$ mea explain.json
```
